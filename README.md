# Ticket Marketplace v18

## Deploy
1. Subir el contenido de esta carpeta a GitHub.
2. Vercel detecta Vite: `npm run build`, output `dist`.
3. Variables:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_KEY

## Supabase
Ejecutar `supabase/schema-v18.sql` en SQL Editor.
